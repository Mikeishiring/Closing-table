üéØ Key Principle #1: Perfect Centering

Everything shares the same center by using inset: 0 with flexbox centering.

The Magic Formula: position: absolute; inset: 0; display: flex; align-items: center; justify-content: center;
/* Container - defines size */ .thumb { position: relative; width: 32px; height: 32px; } /* Background - fills entire container */ .thumb-bg { position: absolute; inset: 0; /* Shorthand for top/right/bottom/left: 0 */ border-radius: 50%; background: white; } /* Icon container - also fills entire container */ .thumb-icon { position: absolute; inset: 0; /* Same dimensions = same center */ display: flex; align-items: center; /* Vertical centering */ justify-content: center; /* Horizontal centering */ }
  • Both layers use inset: 0 on the same parent
  • They occupy the exact same 32√ó32px space
  • Flexbox centers the icon within that space
  • Guaranteed pixel-perfect alignment
